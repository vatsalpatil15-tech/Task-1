# CGPA Calculator in C++

## Description
This project is a simple CGPA Calculator developed using C++.  
It allows users to enter course details such as:
- Course name
- Grade point
- Credit hours

The program calculates:
- Total credits
- Total grade points
- Final CGPA

---

## Features
- Input multiple courses
- Store course details
- Calculate total grade points
- Compute CGPA
- Display course-wise information
- Simple console-based interface

---

## Technologies Used
- C++
- Object-Oriented Programming Basics
- VS Code

---

## Formula Used

CGPA = Total Grade Points / Total Credits

Where:

Grade Points = Grade × Credit Hours

---
